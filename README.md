# fileservice 文件服务

    文件服务.

## 目录说明

|  目录 |  文件  |  描述  |
| ----- | ------ | ------ |
| /conf | key.pem,cert.pem | 默认https证书存放位置, 默认没有 |

## 配置清单(默认)

|  KEY  |  默认值  |  可选值  |  描述 |
| ------ | ------ | ---- | ---- |
| `listen.http.address` | 127.0.0.1:8080 | `*` | 对外HTTP服务地址(默认启用) |
| `listen.rpc.address` | 127.0.0.1:8080 | `*` | 对外RPC服务地址(默认不启用) |
